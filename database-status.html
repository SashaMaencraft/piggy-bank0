Важная информация

В настоящий момент проект работает в демонстрационном режиме без подключения к серверной базе данных.

Это означает:

  • ✓ Все данные хранятся только в вашем браузере
  • ✓ При очистке localStorage данные будут безвозвратно удалены
  • ✓ Вход с другого устройства не даст доступа к данным
  • ✓ Администратор видит данные только в админ-панели (локально)

Как работает хранение:

Сохранение
Данные пишутся в localStorage браузера
Доступ
Только текущий браузер имеет доступ
Синхронизация
Не поддерживается между устройствами
Удаление
При очистке браузера данные пропадают
