## 🏦 Что такое «Копилка»?

**Копилка** — это веб-приложение для постановки и отслеживания финансовых целей.  
Вы создаёте «копилки» (цели), пополняете их, следите за прогрессом, получаете достижения и можете проходить верификацию для подтверждения личности.

### 🎯 Цель копилки:
- Установите **конкретную сумму**, которую хотите накопить.
- Выберите **валюту** (RUB, USD, EUR, CNY, GBP, JPY, KZT, BYN, UAH, TRY).
- Установите **дедлайн** — инструмент сам рассчитает, сколько нужно откладывать в день/неделю/месяц.
- Отслеживайте **прогресс** через наглядные графики и уведомления.

### ✨ Основные функции:
- 🔐 Регистрация и вход (данные хранятся в `localStorage`, БД временно отключена)
- 🎨 9 цветовых тем интерфейса и индивидуальные цвета для каждой копилки
- 💱 10 валют с конвертацией при переводе между копилками
- 🏆 20+ достижений (ачивок) с категориями
- 👥 Публичные/приватные копилки и поиск пользователей
- 📸 Загрузка аватаров и обложек профиля
- ✅ Верификация с загрузкой фото лица и паспорта (проверяется администратором)
- 🔊 Звуковые эффекты (можно отключить)
- 📱 Адаптивный интерфейс + отдельная версия для ПК
